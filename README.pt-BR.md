# OpenClawRun (Português - Brasil)

[← Seletor de idioma](./README.md)

OpenClawRun é um app de menu bar para macOS que permite gerenciar o OpenClaw Gateway sem comandos de terminal.

## Prévia do menu

![OpenClawRun menu](./assets/menu-screenshot.png)

## Política de distribuição
- Este repositório é apenas para **distribuição e suporte**
- O código-fonte é fechado (closed source)
- Os binários são publicados em [Releases](../../releases)

## Download
1. Abra [Releases](../../releases)
2. Baixe o `OpenClawRun-*.dmg` mais recente
3. Arraste o app para Applications

## Reportar problemas
- [Issues](../../issues/new/choose)
- Informações úteis:
  - versão do macOS
  - versão do OpenClawRun
  - versão do OpenClaw (`openclaw --version`)
  - saída de `Copy Diagnostics` no OpenClawRun

## Requisitos
- macOS 14+
- Gateway padrão do OpenClaw: `127.0.0.1:18789`
