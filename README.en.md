# OpenClawRun (English)

[‚Üê Language chooser](./README.md)

OpenClawRun is a macOS menu bar app to manage OpenClaw Gateway without terminal commands.

## Distribution policy
- This repository is for **distribution and support** only
- Source code is closed source
- Binaries are distributed via [Releases](../../releases)

## Download
1. Open [Releases](../../releases)
2. Download the latest `OpenClawRun-*.dmg`
3. Drag the app into Applications

## Report issues
- [Issues](../../issues/new/choose)
- Helpful details:
  - macOS version
  - OpenClawRun version
  - OpenClaw version (`openclaw --version`)
  - `Copy Diagnostics` output from OpenClawRun

## Requirements
- macOS 14+
- OpenClaw Gateway default: `127.0.0.1:18789`
